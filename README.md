# Солнечная система на Pygame

Анимация солнечной системы, созданная с помощью Pygame. Планеты, спутники и астероиды вращаются вокруг Солнца.

## Как запустить

1. Установи Python (если ещё не установлен)
2. Установи Pygame: pip install pygame
3. Скачай и запусти:
   git clone https://github.com/dianaLoki/SolarSystemGraphics.git
   cd SolarSystemGraphics
   python main.py

## Что внутри

- Солнце в центре
- 6 планет (Меркурий, Венера, Земля, Марс, Юпитер, Сатурн)
- Спутники у Земли, Юпитера и Сатурна
- Пояс астероидов
- Звёзды на заднем фоне
